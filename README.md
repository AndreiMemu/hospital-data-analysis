# Hospital Patient Data Analysis

Exploratory data analysis of a real-world hospital patient dataset using Python and pandas.

## Questions Answered
- Which diagnosis is most common overall?
- Which diagnoses are most common by age group (children, teenagers, young adults, adults, seniors)?
- Which condition keeps patients hospitalised the longest?
- Which hospitals see the most patients?

## Key Findings
- Cholelithiasis (gallstones) is the most common diagnosis overall
- Asthma dominates in children, Acute Bronchitis in teenagers
- Young adults are most affected by Cholelithiasis, linked to weight changes and hormonal factors in women
- Adults show rising rates of Atrial Fibrillation — a serious heart condition
- Hospital stay duration is relatively uniform across diagnoses (5-6 days average), suggesting standardised discharge patterns
- HOSP-25 sees the highest patient volume with 72 patients

## Dataset
Real-world hospital patient dataset — 5000 rows, 7 columns covering patient ID, age, gender, diagnosis, admission date, discharge date and hospital ID.
